* conda install tensorflow=2.1

* conda isntall tensorflow=2.1 -> tensorflow 2.1 버전 (cpu)설치 
* conda list // 패키지 설치 한 list 보기 
* pip freeze // pip 로 설치한 패키지 list 보기
